    Function IODev.seekCurrent

    Seek the stream. seekCurrent seeks from the current stream position, seekAboslute seeks to the given position offset from the beginning of the stream, and seekEnd seeks to the position offset bytes from the end of the stream (backwards).

    ulong seekCurrent (
      long offset
    );

    Note, this throws an exception if seeking fails or isn't supported.

    params

    offset = bytes to seek.

    returns

    The position of the stream from the beginning of the stream after seeking, or ulong.max if this cannot be determined.
